# 标题
Markdown用`#空格`来表示标题，最多6级标题，语法如下：
```
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```
显示效果：  
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

# 缩进、换行、空行、对齐方式
## 缩进
```
&emsp;或&#8195; //全角
&ensp;或&#8194; //半角
&nbsp;或&#160;  //半角之半角
```
## 换行
由于markdown编辑器的不同,可能在一行字后面，直接换行回车，也能实现换行，但是在Visual Studio Code上，想要换行必须得在一行字后面空两个格子才行。
## 空行
空一行或者空多行都表示一个空行；
## 对齐方式
语法如下：
```
<center>行中心对齐</center>
<p align="left">行左对齐</p>
<p align="right">行右对齐</p>
```
显示效果:  
<center>行中心对齐</center>
<p align="left">行左对齐</p>
<p align="right">行右对齐</p>

# 超链接、注脚、锚点
## 显式超链接
显示超链接只需要用<>包起来， Markdown 就会自动把它转成链接地址：
```
<http://www.baidu.com>
```
显示效果：  
<http://www.baidu.com>

## 隐式超链接
[]里写链接文字，()里写链接地址, ()中的""中可以为链接指定title属性，title属性可加可不加。title属性的效果是鼠标悬停在链接上会出现指定的title文字，链接地址与title前有一个空格。
语法如下：
```
[百度一下](http://www.baidu.comm "百度")
```
显示效果：  
[百度一下](http://www.baidu.comm "百度")

## 引用超链接
引用超链接一般用在某个文档中多次使用同一个链接地址，在这种情况下，可以预先在文档尾部定义超链接，然后在文档中引用预先定义的超链接：
```
门户网站哪家强？[百度][1]、[新浪][2]。

[1]:http://www.baidu.com 
[2]:http://www.sina.com
```
显示效果：  
门户网站哪家强？[网易][1]、[新浪][2]。

[1]:https://www.163.com/
[2]:http://www.sina.com

## 注脚
注脚是对内容一种注解说明方式，其语法如下：
```
使用Markdown[^1]可以高效的书写文档, 还可以直接转换成HTML[^2]。

[^1]:Markdown是一种纯文本标记语言

[^2]:HyperText Markup Language 超文本标记语言
```
其中注脚使用`[^注脚名称]`来定义，然后在需要的地方使用`[^注脚名称]`，注脚一般定义在文档的尾部；

显示效果：  
使用Markdown[^1]可以高效的书写文档, 还可以直接转换成HTML[^2]。

[^1]:Markdown是一种纯文本标记语言

[^2]:HyperText Markup Language 超文本标记语言

## 锚点
锚点的定义语法如下：
```
[锚点](#锚点ID)

<a id="锚点ID">去锚点</a>
```
显示效果：  
[锚点](#锚点ID)

<a id="锚点ID">去锚点</a>

# 无序列表、有序列表
## 无序列表 
使用 *，+，- 都可以表示无序列表：
```
* 无序列表项一
+ 无序列表项二
- 无序列表项三
```
其中符号后面要追加空格；

显示效果：  
* 无序列表项一
+ 无序列表项二
- 无序列表项三

## 有序列表
有序列表则使用数字接着一个英文句点。
```
1. 有序列表项一
2. 有序列表项二
3. 有序列表项三
```
其中句点后面要追加空格；

显示效果：  
1. 有序列表项一
2. 有序列表项二
3. 有序列表项三

# 插入图片
插入图片的语法如下：
```
![图片ALT提示](images/icon.jpg "图片标题")
```
其中`图片ALT提示`指图片不能显示时的提示文字，`图片的地址`可以是相对路径，也可以是网络图片地址，`图片标题`是指鼠标悬停时的提示文字，可以省略；

显示效果：  
![图片ALT提示](images/icon.jpg "图片标题")



# 表格
表格的语法如下：
```
name | 111 | 222 | 333 | 444
:-: | :-: | :-: | :-: | :-:
aaa | bbb | ccc | ddd | eee
fff | ggg| hhh | iii | 000
```
其中符号的意义：  
1. |、-、:之间的多余空格会被忽略，不影响布局。
2. 默认标题栏居中对齐，内容居左对齐。
3. -:表示内容和标题栏居右对齐，:-表示内容和标题栏居左对齐，:-:表示内容和标题栏居中对齐。
4. 内容和|之间的多余空格会被忽略，每行第一个|和最后一个|可以省略，-的数量至少有一个。

显示效果如下：
name | 111 | 222 | 333 | 444
:-: | :-: | :-: | :-: | :-:
aaa | bbb | ccc | ddd | eee
fff | ggg| hhh | iii | 000
