## 透明度
opacity透明度属性，取值范围0-1，透明度作用于整个元素，不只是背景；  
例如，创建一个有透明度的图片：
```
img
{
  opacity:0.4;
  filter:alpha(opacity=40); /* IE8 及其更早版本 */
}
```
Opacity属性值从0.0 - 1.0。值越小，使得元素更加透明。  
IE8和早期版本使用滤镜：alpha（opacity= x）。 x可以采取的值是从0 - 100。较低的值，使得元素更加透明。
