语义标签都可以用div+class来代替，只是语义标签的可读性更好一些；
```
<body>
    <header>
        <nav>
            <a href="#">首页1</a>
            <a href="#">首页2</a>
            <a href="#">首页3</a>
            <a href="#">首页4</a>
        </nav>
    </header>
    <section>
        <article>
            我是文章主体
        </article>
        <aside>
            <a href="#">我是一个链接1</a>
            <a href="#">我是一个链接1</a>
            <a href="#">我是一个链接1</a>
            <a href="#">我是一个链接1</a>
        </aside>
    </section>
    <footer>
        我是一个页脚
    </footer>
</body>
```
其中标签解释如下：
- header：页眉
- section：区块
- footer：页脚
- nav：导航
- article：文章主体
- aside：侧边栏
