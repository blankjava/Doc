## 标题标签
h标签为标题标签，分为6个等级，每个p标签都占一整行：
```
<body>
    <h1>标题1</h1>
    <h2>标题2</h2>
    <h3>标题3</h3>
    <h4>标题4</h4>
    <h5>标题5</h5>
    <h6>标题6</h6>
</body>
```

## 段落标签
p标签为段落标签，占一整行：
```
<body>
    <p>段落标签1</p>
    <p>段落标签2</p>
</body>
```

## 换行标签
br标签为换行标签：
```
<body>
    <font>hello</font>
    <br>
    <font>world</font>
</body>
```

## 水平线标签
hr标签为水平线标签：
```
<body>
    <font>hello</font>
    <hr>
    <font>world</font>
</body>
```
